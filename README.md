Coffee Sales Dashboard (Power BI)
Overview
This project is my first end-to-end Power BI dashboard, built to analyze coffee shop sales performance across time, products, locations, and customer activity.
The goal of the project was to practice:
data modeling
KPI design
interactive visuals
basic analytical storytelling using Power BI
The dashboard focuses on operational and exploratory analysis, not executive-level reporting.
Dashboard Summary
Key metrics and insights included in the dashboard:
Total Sales, Orders, and Quantity Sold
Month-over-month performance comparison
Weekday vs Weekend sales split
Sales by store location
Sales trends over time
Product category and product-level performance
Hourly and daily sales patterns
A static PDF export of the dashboard is included for quick viewing without Power BI Desktop  .
